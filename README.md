# Tateti en Python 🎮

Este es un juego de Tateti (Tic Tac Toe) hecho en Python desde la terminal.

## Características

* Jugador vs jugador
* Validación de movimientos
* Detección de ganador
* Empate
* Marcador de puntos

## Cómo jugar

1. Ejecutar el archivo:

```
python tateti.py
```

2. Elegir la posición del 1 al 9.
